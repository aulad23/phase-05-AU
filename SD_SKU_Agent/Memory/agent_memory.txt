AGENT MEMORY
=============
এখানে Agent সব কিছু জমা রাখে।
নতুন vendor করলে এখান থেকে pattern নেয়।

==============================================
GLOBAL RULES (সব vendor-এ apply হবে)
==============================================

[RULE 1] PRODUCT FAMILY ID
---------------------------
Product Family Id = Product Name থেকে প্রথম special character-এর আগের অংশ।
Special characters: , . _ - @
উদাহরণ:
  "Aulad_Hossain"               -> "Aulad"
  "South Hampton Table, Sable"  -> "South Hampton Table"
  "Ribbon Cocktail Table-Round" -> "Ribbon Cocktail Table"
  "Cyrus Hall Table"            -> "Cyrus Hall Table"  (no special char = full name)
Code:
  import re
  product_family_id = re.split(r'[,._\-@]', product_name)[0].strip()

[RULE 2] ALL CATEGORIES
------------------------
Vendor name দিলে সব category scrape করতে হবে — partial না।
DEMO_MODE=True হলে 5 product, False হলে সব।

[RULE 3] OUTPUT FORMAT (reference: Allan Knight.xlsx)
------------------------------------------------------
Row 1 : Brand | [Vendor Name]
Row 2 : Link  | [Base URL]
Row 3 : (empty)
Row 4 : Headers
Row 5+: Data
সব category একটাই Excel file-এ।

[RULE 4] SHOPIFY SITES
-----------------------
Shopify site হলে product.json API ব্যবহার করো:
  GET [product_url].json
  SKU      -> variants[0].sku
  Price    -> variants[0].price
  Desc     -> body_html (strip HTML tags)
  Img      -> images[0].src
  Family   -> product_type (fallback: Rule 1)

[RULE 5] COLUMNS (fixed order then dynamic)
--------------------------------------------
FIXED (always in this exact order):
  Manufacturer, Source, Image URL, Product Name, SKU, Product Family Id,
  Description, Dimension, Weight, Width, Depth, Diameter, Length, Height

DYNAMIC (add after Height, only if data exists on the site):
  Seat Width, Seat Depth, Seat Height, Arm Height, Arm Width,
  List Price, Material, Finish, Color, Style, Collection,
  ... any other field found on the page

Rule: dynamic columns আসে website থেকে — vendor-এ যা available তাই নেবে।
"Dimension" column = full raw dimension string (e.g. "54 x 36 x 17 H")

[RULE 6] OUTPUT FILE NAME
--------------------------
Demo : Agent/Demo/[VendorName]_demo.xlsx
Full : Agent/Data/[VendorName]/[VendorName].xlsx
Format: Row1=Brand|Name, Row2=Link|URL, Row3=empty, Row4=headers, Row5+=data

[RULE 7] VENDOR DECISION (from Vendor List)
--------------------------------------------
Vendor List Excel: Agent/Vendor_List/SD_Web Scraping - Status Tracker.xlsx
Columns: Vendor, Category, Rank, Sample Link, Scraping Status, Remarks
Agent workflow:
  1. Vendor name দিয়ে search করো (case-insensitive)
  2. Scraping Status check করো
  3. URL নাও Sample Link থেকে
  4. Rank দেখে priority বোঝো (1=highest)
  5. Remarks থাকলে সেটা note করো

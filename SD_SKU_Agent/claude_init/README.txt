CLAUDE INIT — AGENT CHAT LOG FOLDER
=====================================
এখানে Agent নিয়ে সব Claude conversation জমা থাকে।

FORMAT:
  [DATE]_[VendorName]_chat.txt  ← Vendor-specific chat
  [DATE]_general_chat.txt       ← General agent instruction chat

HOW TO USE:
  - Agent যখন কোনো vendor নিয়ে কাজ করে, সেই conversation এখানে save হয়
  - পুরনো instruction দেখতে এখানে আসুন
  - নতুন instruction দিতে Chat/ folder ব্যবহার করুন

NAMING EXAMPLE:
  2026-04-19_ZUO_chat.txt
  2026-04-19_general_chat.txt
